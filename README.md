# servodriver


ServoDriver is a 3-servo easy-to-use servo driver board. It operates over UART. 

I made ServoDriver because I wanted to not only introduce myself to the robotics utility field but also to create a useful servo driver solution that can be controlled via a laptop or SBC. I also made this to learn how to make a PCB without a dev board.
